# Bank-Analytics-Dashboard-Analysis
Built an interactive Bank Loan Analytics Dashboard in Excel analyzing 65K+ loan records. Performed data cleaning, KPI calculations, pivot analysis, and created dynamic charts with slicers. Analyzed loan performance, repayment trends, borrower demographics, default rates, and branch-wise revenue for business insights.


Introduction

The Bank Analytics Dashboard project focuses on analyzing loan portfolio performance, borrower behavior, repayment trends, and risk distribution using Microsoft Excel. The purpose of this project is to provide meaningful insights into loan disbursement, collections, delinquency, and customer demographics through an interactive dashboard. The dashboard helps financial institutions make data-driven decisions for better loan portfolio management.

Business Requirements

The dashboard was developed based on the following key business questions:

What is the total amount of loans funded?
How many total loans are active in the portfolio?
What is the total amount collected from borrowers?
What is the total interest earned?
Which branches are generating the highest revenue?
Which states have the highest loan concentration?
What are the most common loan product categories?
What is the borrower age group distribution?
What is the current loan status distribution?
How many loans are delinquent or defaulted?
What is the overall loan maturity structure?
What are the yearly loan disbursement trends?

Key Performance Indicators (KPIs)

The following KPIs were identified and implemented in the dashboard:

Total Loan Amount Funded: Total value of loans disbursed.

Total Loans: Total number of loan accounts.

Total Collection: Total repayment collected from customers.

Total Interest: Total revenue earned through interest.

Default Loan Count: Number of defaulted loans.

Delinquent Client Count: Number of borrowers with overdue payments.

Default Loan Rate: Percentage of default loans.

Delinquent Loan Rate: Percentage of delinquent loans.

No Verified Loans: Number of unverified loan accounts.

Loan Maturity: Loan term distribution between short-term and long-term loans.

Data Cleaning and Transformation

Before building the dashboard, the following data preparation steps were performed:

Handled Missing Values: Checked and managed missing data in important columns.

Standardized State Names: Corrected incorrect state entries.

Validated Branch Names: Standardized inconsistent branch naming.

Checked Duplicate Records: Verified uniqueness of Account ID.

Created Derived Columns:

Age_T for analytical grouping

Interest Rate Percentage for easier interpretation

Retained Business-Based Nulls: Fields like Closed Date and Loan Transfer Date were left blank where applicable.

Results & Business Impact

The dashboard provided important business insights:

Loan Concentration:

Punjab and Haryana had the highest loan distribution.

Product Performance:

Home loans and service-related loans formed the largest portion of the portfolio.

Risk Analysis:

Default loan rate was 1.55%, while delinquent loan rate was 10.84%.

Customer Demographics:

Most borrowers were in the 26–45 age group.

Loan Maturity:

The majority of loans were 36-month loans.

Branch Performance:

Certain branches contributed significantly higher revenue.
